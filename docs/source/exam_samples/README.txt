Place sample exam PDF files here — one per course.

Naming convention: include the paper code in the filename.
Examples:
  2A1_sample_exam.pdf
  2A2_sample_exam.pdf
  2B1.pdf

The generate_questions.py script reads these to calibrate question phrasing,
distractor style, and difficulty for each course.

If no PDF is found for a course, a generic Power Engineering style guide is used.
